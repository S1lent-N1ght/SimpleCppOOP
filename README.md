# SimpleCppOOP
This is for beginners who are following OOP structure. 
The task: Allow the user to create an account, a username and a 'password'.

You must follow OOP structure and comment your code and reason behind what you are doing. 
Use:
Main.cpp
Header.h

####
- Follow OOP Structure
- Include Comments
- -
-   Use Main.cpp and Header.h

- Password Validation
